This is an implementation of our own game called "Connect the Dots" using pygame module

**To run the code, open Terminal and type the commands:**
```
cd Desktop
```
```
git clone https://github.com/shahidshaiksk/Connect-the-Dots.git
```
```
cd Connect-the-Dots
```
```
pip install pygame
```
```
python main.py
```
